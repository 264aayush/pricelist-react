Steps to run this file on your local system.

1.) Make sure you have node package manager installed.
2.) open terminal and navigate to the project folder.
3.) Run "npm install" command in terminal to install all dependencies.
4.) To run the project npm start.
5.) By default It will run on https://localhost:3000 . Unless you have another app running on port 3000. Then you can check on terminal to see which port it is running on.
